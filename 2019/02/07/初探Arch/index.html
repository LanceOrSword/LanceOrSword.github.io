title:初探Arch

arch

使用工具：

Oracle VM VirtualBox

1.新建虚拟机 12GB(可能真的太大了，不过是动态分配，不会真12GB全占)
2.载入启动盘 毫无疑问，archLinux.iso

3.选择第一个,等待，成功进入命令行界面
4.打开浏览器,找教程
5.cfdisk - dos分区sda1、sda2,sda2要设置成bootable

  1. 挂载sda1到Linux根目录下 : mount /dev/sda1 /mnt

8.设置软件源

grep 163 /etc/pacman.d/mirrorlist > bak

cat bak > /etc/pacman.d/mirrorlist

pacman -Syy

9.安装基本系统 : pacstrap /mnt base base-devel
10.起来喝口水
11.生成fstab : genfatb -U -p /mnt >> /mnt/etc/fstab
12.配置基础系统 : arch-chroot /mnt /bin/bash
13.设置locale
13.1 locale语言 : nano /etc/locale.gen
13.1.1 去掉en_US.UTF-8和zh_CN.UTF-8前的#
13.1.2 CTRL+X
13.1.3 Y
13.2 locale时间 : ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime then hwclock –systohc –utc
14.修改主机名 echo ___ > /etc/hostname
15.设置自动连接网络 systemctl start dhcpcd then systemctl enable dhcpcd
16.设置root密码 passwd
17.安装qrub 

pacman -S grub os-prober

grub-install --target=i386-pc --recheck /dev/sda

grub-mkconfig -o /boot/grub/grub.cfg

18.卸载分区，重启机器 exit then umount -R /mnt then reboot
19.卸掉引导盘
